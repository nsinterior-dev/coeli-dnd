# coeli-dnd

Welcome to **coeli-dnd** - an improved drag-and-drop library for React and Next.js with advanced nested list support!

## Why coeli-dnd?

Built to solve complex nested drag-and-drop scenarios that are challenging with existing libraries. If you've struggled with nested lists where items and groups coexist in the same array, this library is for you.

- **üéØ Nested Drag & Drop** - Seamlessly handle items and groups in the same array

## The 6 Core Operations

1. **Reorder groups** - Move groups around the canvas
2. **Reorder mixed items** - Drag items/groups in mixed arrays
3. **Item to group** - Move items from root level into groups
4. **Item from group** - Move items from groups to root level
5. **Reorder in group** - Rearrange items within a group
6. **Item between groups** - Move items from one group to another

## Quick Start

```jsx
import {
  CoeliDndProvider,
  useCoeliDndContext,
  DraggableDroppable,
} from 'coeli-dnd';

const items = [
  { id: '1', label: 'Item 1', type: 'ITEM' },
  {
    id: '2',
    label: 'Group 1',
    type: 'GROUP',
    children: [
      { id: '3', label: 'Item 2', type: 'ITEM', parentId: '2' },
    ],
  },
];

function Canvas() {
  const { items } = useCoeliDndContext();

  return (
    <div>
      {items.map(item => (
        <DraggableDroppable key={item.id} item={item}>
          <div>{item.label}</div>
        </DraggableDroppable>
      ))}
    </div>
  );
}

export default function App() {
  return (
    <CoeliDndProvider initialItems={items}>
      <Canvas />
    </CoeliDndProvider>
  );
}
```

## Data Structure

Items and groups share the same structure:

```typescript
interface DndItem {
  id: string | number;
  label: string;
  type: 'ITEM' | 'GROUP';
  children?: DndItem[];
  parentId?: string | number | null;
  [key: string]: any; // Additional custom properties
}
```
